# Title
This is the script and table depository of the manuscript  

**Estimating the distribution of fitness effects of loss of heterozygosity (LOH) events using an engineered library of Saccharomyces cerevisiae**  
*Yi-Hong Ke, Michelle Orozco-Quime, Josh Bauman, Tamilie Carvalho, Shuhua Ge, Anuj Kumar, Timothy Y. James*  

# scripts/
yeastLOH_pub.md: main workflow and commands for genome analysis  
scripts/metadata.R: parse and consolidate strain information  
scripts/barseq_pub.R: main statictical analyses and plotting R code  
scripts/analyze_DFE.py: MLE for whole distribution  
scripts/analyze_DFE_tail.py: MLE for tail distribution  
scripts/plotDFE.py: plotting results from analyze_DFE.py and analyze_DFE_tail.py  
scripts/do_diptest.py: Hartigan's dip test for bimodality in DFE  
scripts/LOH_detect.py: define LOH regions in each strain from the WGS .vcf file  

# tables/
tables generated by scripts  

# plots/
plots generated by scripts  

# pub_tables/
tables of metada or curated manually  

# bartender/
barcode frequency counting by *bartender*  

# LOH_detect/
.bed files for LOH regions in each genome and .bedgraph files for LOH region coverage across strains  

# ref/
reference genome and sequences
